# 🇺🇿 O'zbekiston To'lov Balansi Dashboard

Interaktiv to'lov balansi tahlil dashboard'i. IMF BPM6 metodologiyasi asosida.

## Xususiyatlar

- 📊 25+ grafik turlari (area, waterfall, heatmap, scatter va boshqalar)
- 🌐 3 tilli: O'zbek / English / Русский
- 📈 Choraklik va yillik ma'lumotlar
- 📥 CSV va Excel formatida yuklab olish
- 🎨 Zamonaviy, professional dizayn

## Manba

O'zbekiston Respublikasi Markaziy banki statistikasi (2005–2025)

## Texnologiyalar

- Python + Streamlit
- Plotly (grafiklar)
- Pandas (ma'lumotlar qayta ishlash)
